How this works. Each goal is reconstructed from event data on a 120×80 pitch (the StatsBomb open-data schema): every pass, carry and the finishing shot, animated in sequence so you can read the build-up. Drop a real StatsBomb match events file into GOALS and it plays as-is.
Live note. Free live feeds for WC2026 (football-data.org) give only score, scorer and minute — not player positions — so a true positional replay of a live 2026 goal needs StatsBomb (historical) or a paid positional feed. Until then, live goals show as a scorer/minute timeline; finished matches with event data get the full reconstruction. The match rail pulls wcpa26.com/api/fixtures.json when served online, and falls back to bundled samples offline.
